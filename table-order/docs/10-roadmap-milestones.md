# 10. 로드맵 & 마일스톤

- 버전: v0.1 (2026-07-12)
- **이 문서의 체크박스가 프로젝트 진행 상태의 SSOT다.** 오케스트레이터는 마일스톤 마감 시(CLAUDE.md 프로토콜 ⑦) 여기를 갱신·커밋한다.
- 기간은 에이전트 오케스트레이션 기준 캘린더 추정(사람 리뷰 대기 포함). 총 **7~8주 → 베타**.

| 산출물 | 제공자 비교 PoC 보고(docs/12 §7)와 어댑터 확정, 생성 잡 파이프라인(QUEUED→READY→SELECTED/FAILED 환불), 프롬프트 템플릿 3종(HERO/DECONSTRUCTED/CLOSEUP), 후보 선택·라벨 에디터, 크레딧 차감·환불, AI 생성 고지 배지 |
| DoD | A-6(재료 입력→적용 3분 내) 리허설 통과, 미검수 노출 차단(I-8) 회귀 그린, 데모 매장에 AI 분해컷 1종 적용·스크린샷, 생성 p95 < 60s |

## 리스크 & 대응

| 리스크 | 영향 | 대응 |
|---|---|---|
| 룩북 '감성'이 사진 품질에 의존 — 매장 사진이 나쁘면 차별점 붕괴 | 높음 | 업로드 가이드·자동 보정(밝기/크롭 제안) 백로그 상향, 베타 매장은 촬영 지원 패키지로 선별 |
| Supabase Realtime 동시 연결 한도/지연 | 중간 | 폴백 프로토콜(docs/07 §4)이 1급 시민 — 실시간 죽어도 서비스 성립. 부하 테스트 M3 DoD 포함 |
| 선결제 자금 흐름 법적 구조(PG 하위가맹) | 높음 | M4 전 법무 확인(docs/09 §6). 불가 시 MVP는 후불 전용으로 출시(모드 A만) |
| 피크타임 태블릿 브라우저 절전/백그라운드로 알림 누락 | 중간 | Wake Lock+재알림+미확인 배지, 상주 모드 가이드(docs/06 §8) |
| 에이전트 병렬 작업 간 계약 드리프트 | 중간 | 계약 단독 커밋 원칙 + 계약 테스트(docs/04 §5)가 게이트 |
| 가격 가설 미검증 | 중간 | 베타 3매장 무료 + 지불의사 인터뷰 후 확정 |
| AI 연출컷 품질 편차·실물 오인(표시광고 리스크) | 중간 | 검수 승인제(I-8) + 'AI 생성 이미지' 고지 배지 + 프롬프트 템플릿 고정. 품질 미달 시 분해컷 한정으로 스코프 축소 |
| AI 생성 비용 폭증 | 낮음 | 플랜별 크레딧 한도·실패 환불·rate limit, 생성 단가 상한 모니터링(docs/12 §7) |
