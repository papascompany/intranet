## 1. 시장 구조와 우리의 자리

한국 테이블오더 시장은 **전용 태블릿 진영**(티오더·메뉴잇·페이히어 오더 등)이 주도한다. 이들의 강점은 통신사 결합 영업망(KT 하이오더 등)·광고/프로모션 수익으로 낮춘 단말 부담·오프라인 AS이고, 구조적 약점은 ① 테이블당 하드웨어 원가와 장기 약정(통상 수년으로 알려짐 — 검증 필요) 때문에 **소형 매장에서 경제성이 나오지 않는 것**, ② 가로 거치 공용 태블릿이라 '개인 폰 세로 화보' 경험을 줄 수 없는 것, ③ 광고 수익 모델과 '매장 브랜드 지면'이 충돌하는 것이다.

한편 "국내 홀에서 QR이 태블릿에 밀린 선례"(네이버 QR 테이블주문 종료로 알려짐 — **사실 검증 필요**)가 존재한다. 교훈은 'QR 방식 자체의 패배'가 아니라 **세그먼트 선택의 문제**로 읽는다: 회전율 중심 대형 매장에서 QR은 태블릿을 못 이긴다. 우리는 반대편 — 브랜딩이 무기이고 태블릿 경제성이 안 나오는 매장 — 에서 시작한다.

**우리의 자리**: 8~20테이블 · 태블릿 미도입 · 사진/브랜딩 투자형 매장. 판매하는 것은 '주문 단말'이 아니라 **브랜드 경험 + 객단가 상승 + 하드웨어 0원 당일 개통**.

## 2. 비치헤드와 GTM 단계

| 단계 | 내용 | 성공 기준 |
|---|---|---|
| **β (M6, 3매장)** | 비치헤드: '인스타 브랜딩 투자형 브런치/베이커리 카페, 8~20테이블, 태블릿 미도입' × 단일 상권 — **확정: 본선 연남·연희, 백업 성수**(스코어링·전환 게이트는 docs/14 §1). 촬영 지원 패키지로 선별 모집 | 3매장에서 **세션당 추가주문율 ≥ 25% 실측**(측정 정의 docs/14 §5.2) + 오너 NPS 인터뷰. **실행 플레이북: docs/14** |
| **1단계 (출시~3개월)** | 같은 상권 밀도 공략 20매장 — 채널: 인스타 DM 아웃바운드 100곳 → 미팅 20 → 계약 10(가설, 실측으로 교정), 베타 사례 페이지가 소재 | 유료 10매장, 무료→유료 전환 ≥ 8%/90일 |
| **2단계 (3~9개월)** | 상권 복제 + 업종 확장(와인바·다이닝 — 캐치테이블 오더 인접 영역이므로 후순위 진입) + FREE 플랜 PLG(룩북 하단 배지 → 옆 매장 오너가 손님으로 경험) | MRR ₩300만, 월 이탈 < 5% |

**촬영 지원 패키지(베타·1단계 무기)**: 반나절 출장 촬영(메뉴 10종) + 룩북 세팅 대행 + PRO 3개월 무료. 매장당 원가 상한 40만원. 조건 = 사례 공개 동의 + QR 스탠드 상시 부착. 이후 **단품 유료 상품화**(예: 29만원 — §6 수익원 ④).

**플라이휠**: 촬영 패키지 → 화보급 사례 → 사례 페이지·인스타 릴스 → 인바운드 → (사진이 없는 매장은) AI 연출컷이 촬영을 대체 → 저CAC 확장.

## 4. 도입 장벽(전환 비용) 대응

최대 장벽은 하드웨어가 아니라 **최초 메뉴·사진 입력**이다.

1. M5: 온보딩 CSV 일괄 임포트 + 업종별 카테고리 템플릿 (docs/09 §2)
2. 백로그 최상위: 종이 메뉴판/배민 화면 **사진 → 비전 AI 초안 추출**(이름·가격·설명) → 검수 → 일괄 생성. 목표 **'메뉴 30개, 30분 내 이관'**. 사진 없는 메뉴는 AI 연출컷 제안 배지로 연결(docs/05 §9)
3. 기존 포스 병행 마찰 제거: 일 마감 대사 리포트·정산 기록 생략 모드(docs/06) + '주문=MENUBOOK, 결제·세무=기존 포스' 역할 분담 1페이지 가이드(온보딩 산출물)

## 6. 수익원 로드맵

1. **구독**(주 수익 — docs/09 §3 사다리)
2. **AI 크레딧 애드온**: 10회 ₩9,900 (원가율 ≤ 30%)
3. **촬영·룩북 세팅 대행 상품화**: 단품 29만원(가설) — GTM 패키지의 수익 전환
4. (검토) 모드 C(테이블 QR 셀프 정산) 도입 시 **매장 직계약을 유지하는 PG 파트너 리베이트** — 자금이 플랫폼을 경유하지 않는 구조를 유지해야 하므로 법무 검토 항목(docs/09 §6)에 추가. 검토 후 기각하더라도 근거를 이 문서에 기록한다.
5. 광고는 하지 않는다 — '광고 없는 브랜드 지면'이 포지셔닝의 일부다.

## 7. KPI 트리 (North Star: 월 활성 주문 수)

```
월 활성 주문 수
├─ 매장 수: 가입 → 온보딩 완주(메뉴10+사진 ≥60%) → 첫 실주문(≥40%, ≤2일) → 유료 전환(체험 25% / 무료 8%)
├─ 매장당 주문: QR스캔→첫주문 전환 ≥55% (설계 예산 N-10이 담보)
│                └─ 세션당 추가주문율 ≥25% ('한 번 더'·다시 주문 스트립·객단가 위젯)
└─ 리텐션: 월 이탈 <5% ← 선행지표: 통계 월 1회+ 조회 오너 ≥70%, 추가주문 기여액 위젯(ROI 증명)
```
